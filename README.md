
# Encriptador de Texto Alura

El encriptador es una interfaz que proporciona al usuario la facilidad de ingresar un texto determinado para encriptar o desencriptar según las normas de conjugación con las vocales. Además de proporcionar un cuadro de texto que nos permite copiar lo realizado.
